# We're travelers — Parques alrededor del mundo

Página web grupal sobre **parques temáticos y naturales** en distintos países. Combina **HTML**, **CSS**, **JavaScript** y **Bootstrap** para layout responsive y componentes visuales.

**Autora del repositorio:** [@Juli21v](https://github.com/Juli21v)

## Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
- Bootstrap

## Contenido

- Información y presentación de parques por regiones o temáticas
- Trabajo en equipo: reparto de secciones y estilos coherentes

## Cómo verlo

Abre `index.html` en el navegador o sirve la raíz del proyecto con un servidor estático.

## Para reclutadores

Ejemplo de **colaboración** y uso de un framework CSS (Bootstrap) junto con lógica ligera en JS.
